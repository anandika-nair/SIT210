# Task 5.1P – GUI LED Control using Raspberry Pi

## Overview
In this task, I created a simple graphical user interface (GUI) using Python on Raspberry Pi. The aim of this task is to control LEDs using a user-friendly interface instead of normal switches.

## Hardware Used
- Raspberry Pi
- Breadboard
- 3 LEDs (Living Room, Bathroom, Closet)
- Resistors
- Jumper wires

## Software Used
- Python
- Tkinter (for GUI)
- RPi.GPIO (for controlling GPIO pins)

## How It Works
The GUI has three radio buttons for selecting different rooms:
- Living Room
- Bathroom
- Closet

When a room is selected, the corresponding LED turns ON, and the other LEDs turn OFF. This is done using GPIO pins connected to each LED.

## File
- gui_led.py → main Python code for GUI and LED control
- circuit images

## Conclusion
This task shows how we can use a simple GUI to control hardware components using Raspberry Pi in an easy and user-friendly way.
